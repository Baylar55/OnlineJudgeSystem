# Online Judge System

The AlgoCode is a platform like LeetCode, HackerRank,AlgoExpert and etc that designed for practicing programming problems, and facilitating code submissions and evaluations.

## Features

- **Problem Repository**: Browse through a collection of programming problems covering various topics and difficulty levels.
- **User Authentication**: Secure user authentication system allowing users to register, login, and manage their profiles.
- **Contest Management**: Organize coding contests with custom durations, problem sets, and scoring mechanisms.
- **Code Submissions**: Submit code solutions for problems and receive immediate feedback on correctness and efficiency.
- **Solution Sharing**: Share solution that others view and add comment for this solution.
- **Comprehensive Test Cases**: Each problem is accompanied by a comprehensive set of test cases meticulously designed to validate the correctness and efficiency of your code. Upon submission, your code undergoes rigorous testing, and if all test cases pass successfully, your solution is deemed correct, providing you with confidence and assurance in your coding prowess.



## Technologies Used

- **Frontend**: HTML5, CSS3, Bootstrap, Vanilla JavaScript
- **Backend**: .NET 8,
- **Database**: SQL
- **Code Compiling**: Roslyn Compiler, Monaco Editor 
- **Architecture**: Clean Architecture
- **Patterns**: CQRS, Mediatr
